# Event Planner Web

A web-based event planning platform built with Next.js.

## 🚀 Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- React

## 📌 Current Features (Day 1)

- Map page placeholder
- Create Event page placeholder
- Global navigation bar
- Deployed on Vercel

## 🗺 Pages

- `/` → Map view
- `/create` → Create event form (coming soon)
